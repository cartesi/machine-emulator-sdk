# Cartesi Machine Emulator SDK 

The Cartesi Machine Emulator SDK repository.

## Getting Started

### Requirements

- RISCV64 C/C++ Compiler with support for C++17 (tested with GCC >= 8+).
- GNU Make >= 3.81

### Build

```bash
$ make submodules
$ make build-toolchain-env
$ make
```

Cleaning:

```bash
$ make clean 
or
$ make distclean 
```
